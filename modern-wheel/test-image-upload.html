Test Image Upload for Wheel

Open the application at http://localhost:3000

Steps to test:

  1. Add a new item to the wheel
  2. Click the image icon next to the item
  3. Select an image file
  4. Verify the image thumbnail appears next to the item
  5. Check if the image is displayed on the wheel segment
  6. Spin the wheel to ensure images are properly rendered during rotation

Expected behavior:

  • Image upload dialog opens when clicking the image icon
  • Selected image appears as a thumbnail in the input panel
  • Image is displayed on the corresponding wheel segment
  • Images are properly clipped to fit within wheel segments
  • Images rotate correctly with the wheel when spinning
